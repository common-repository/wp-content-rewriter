WP Content Rewriter is a plugin to which helps you to rewrite an article or content of website and blogs with significant values of word. It provides Grammatically
correct sentences.
If you have requirement to rewrite your content or article to post somewhere, You can easily use this just with few easiest steps.


== Description ==

WP Content Rewriter is a plugin which shows up at front end of website, So that any user or viewer can use this to rewrite their content 
or article. You can use this in page,post and widget also. 


A few steps about this plugin:

*  User can use this in any kind of wordpress site, and theme.
*  user can use this plugin in Page, Post, Widgets.
*  There is not requirement of extra hardware and software to use this plugin.
*  Support Responsive design.

== Installation ==

how to install the plugin and get it working.

e.g.

1. Upload `WP-Content-Rewriter`Folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use this shortcode [wp_content_rewriter].
